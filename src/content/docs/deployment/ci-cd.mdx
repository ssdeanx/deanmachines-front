# CI/CD Integration

This guide explains how to integrate deanmachines AI deployments into your continuous integration and continuous deployment (CI/CD) pipeline.

## GitHub Actions Integration

### Basic Workflow

```yaml
name: Deploy deanmachines AI

on:
  push:
    branches: [ main ]
  pull_request:
    branches: [ main ]

jobs:
  deploy:
    runs-on: ubuntu-latest

    steps:
    - uses: actions/checkout@v4

    - name: Setup Node.js
      uses: actions/setup-node@v4
      with:
        node-version: '20'

    - name: Install deanmachines CLI
      run: npm install -g @deanmachines/cli

    - name: Configure deanmachines
      run: dean configure
      env:
        DEAN_API_KEY: ${{ secrets.DEAN_API_KEY }}

    - name: Deploy
      run: dean deploy --env production
      env:
        DEAN_ENV: production
```

## GitLab CI Integration

```yaml
stages:
  - test
  - deploy

deploy_production:
  stage: deploy
  script:
    - npm install -g @deanmachines/cli
    - dean configure
    - dean deploy --env production
  environment:
    name: production
  only:
    - main
```

## Jenkins Pipeline

```groovy
pipeline {
    agent any

    environment {
        DEAN_API_KEY = credentials('dean-api-key')
    }

    stages {
        stage('Deploy') {
            steps {
                sh 'npm install -g @deanmachines/cli'
                sh 'dean configure'
                sh 'dean deploy --env production'
            }
        }
    }
}
```

await suite.runTests({

## Deployment Strategies

### Blue-Green Deployment

```typescript
const deployment = await client.deploy({
  strategy: 'blue-green',
  version: '1.0.0',
  smoke_test: true,
  rollback: {
    enabled: true,
    threshold: 0.1
  }
})
```

### Canary Deployment

```typescript
const deployment = await client.deploy({
  strategy: 'canary',
  version: '1.0.0',
  phases: [
    { percentage: 10, duration: '1h' },
    { percentage: 50, duration: '2h' },
    { percentage: 100 }
  ]
})
```

   - Secure API keys

3. **Monitoring**
   - Monitor deployments
   - Set up alerts
   - Track metrics
